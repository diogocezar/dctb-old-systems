  Tipo
  Nível
Descrição dos níveis :
1 É o nivel básico, nesse nivel o usuário do site poderá fazer apenas locações. Em resumo é o nivel do cliente. (indisponivel para cadastro)
2 É o nivel de administração intermediário, esse usuário terá acesso a algumas informações do painel de controle. mas não terá acesso total.
3 É o maior nivel do sistema, quem possuir esse nivel terá acesso total à administração do site.
