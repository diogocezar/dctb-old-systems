  * A seguir estão descritos como devem ser montados os formulários do site.

[A primeira variavel indica o NOME do text field e deve ser colocada exatamente como escrita inclusive com
as chaves {}. a segunda variavel é o valor desse textfield e deve ser colocada no valor também exatamente como
escrita.

O nome do formulário também eh uma variavel {formTeste}.

A ação do formulário também eh uma vairavel {actionTeste}.

Para melhor entendimento, vamos fazer uma convenção que já venho usando a muito tempo.

NOME DAS VARIAVEIS :

se a palavra for simples, ou seja tive apenas uma palavra (o que não é muito usado) deve ser escita totalmente em minúsculo
como por Ex: {nome}

Se a palavra for composta, ou seja tiver mais de uma palavra deve ser escrita com a primeira letra minúscula
e as demias letras maiúsculas. Por Ex: {exemploDeVariavel}

E é claro as variaveis devem estar escritas entre chaves {}. ]

COMBOS : Nos combos como de costume se altera o nome do combo, e no lugar das <option> coloca-se apenas uma variavel,
parece estranho, mas as opção são "geradas" então um exemplo :

  <select name="select">
    <option value="oi">oi</option>
    <option value="oi">oi</option>
    <option value="oi">oi</option>
  </select>

Ficaria nos templates assim

  <select name="{comboSelect}">
	{comboSelectOpcoes}
  </select>

Então os combos serão descritos

Combo : {comboSelect}/{comboSelectOpcoes}

# Não existe diferença entre os textfields e os textarea, use-os com coerência, onde achar necessário
  a existencia de um campo maior use o text área, da mema maneira subistituindo os nomes das variaveis.

# Para efetuar o cadastro de itens de uma tabela de n pra n ou seja, por exemplo um produto pode estar em vários pedidos e um pedido pode ter vários produtos.

# Para efetuar esse tipo de cadastro usa-se um combo box contendo os "produtos" e uma listbox contendo os produtos de certo pedido.
  
O nome dessa list box deve ser escrito de forma diferente para que o php entenda a como uma lista.

# Exemplo para o nome de uma list box : list_prod[]

# Entretanto como esse nome DEVE ser uma variavel, pode ter o nome normal de uma variavel {listaProdutos}

# Da mesma forma que o combo suas <option> devem ser subistituidas por uma variavel.

Templates :

	Os templates necessários para o site são :


	- Template principal, onde será exibido alguns produtos principais, busca rápida, menu, e categorias.

	- Template interno, contendo todos os componentes do principal entretando com a parte central limpa, para a inserção de formulários para a administração e eventuais outras áreas do menu.

	- Template produtos lista, onde será listado os produtos da busca, ou produtos escolhido pelas categorias. os produtos deverão conter uma miniatura não muito detalhada, e as informações do produto.

	- Template produtos imagem, idem ao anterior, entretando o destaque não está nas informações e sim na figura do produto mais detalhada.

* As listas serão limitadas com paginação.
