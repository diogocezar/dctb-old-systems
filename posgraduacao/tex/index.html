    O que é LaTeX ?

    TeX     (Pronuncia-se ”tek”) foi criado no final da década de 1970 por Donald Knuth, e se resume em ser um sistema de editoração de textos.TeXoferece um grande controle sobre a estrutura do documento a ser editado e, justamente por isso, traz uma complexidade na edição indesejável à maioria dos usuários comuns.LaTeX é um conjunto de macros escritas para oTeXque permite a criação de textos na mais alta qualidade tipográfica, usando um layout profissional pré-definido. Seu criador foi o Leslie Lamport que brincou com seu sobrenome e com a língua espanhola, e colocou o nome de LaTeX em seu programa. Portanto, o LaTeX nada mais é que uma interface para o TEX, mas obviamente, mais simples de ser utilizada.

    WYSIWYG

    Em um sistema WYSIWYG, como o MS-Word, o autor do documento define o layout através de comandos interativos enquanto digita o texto na tela. O resultado final da formatação é visualizado durante todo o processo. O que é visto na tela é o que será impresso. Os comandos para modificar o layout são bastante intuitivos de compreensão fácil.

    Normalmente os documentos editados com WYSIWYG são verdadeiras obras de arte. Vários tamanhos e faces de fonte diferentes, tamanhos de páginas não-ortodoxos(“Se eu aumentar um pouco o tamanho da página e diminuir o tamanho da fonte, esta seção vai caber em uma página”), e não raramente cores, muitas cores. Obviamente isto não é uma regra absoluta, mas neste tipo de editor, parece ser muito mais fácil fazer a coisa errada. Mesmo aqueles poucos que tem um senso estético apurado são castigados pela (falta de) filosofia destes editores.

    Em LaTeX, o autor não se preocupa com a formatação durante a digitação do texto. O importante é a estrutura lógica, e este é talvez a grande força deste sistema, já que força o autor a criar documentos bem estruturados. Uma vez definidos os capítulos, seções, subseções, citações, fórmulas, bibliografias, etc., entra em cena o formatador, que escolhe, a partir de algumas diretivas, o melhor layout para o documento. O resultado final é um documento com aparência profissional e logicamente consistente.

    Para efeito de comparação, seguem algumas vantagens de LaTeXsobre WYSIWYG (e o oposto).

    Vantagens:

    1. A inclusão de fórmulas matemáticas é suportada de maneira conveniente.
    2. O usuário apenas tem de aprender alguns comandos fáceis de entender que especificam a estrutura do documento. Quase nunca é necessário pensar no layout final do documento.
    3. Estruturas complexas como notas de rodapé, referências, sumários e bibliografias são geradas facilmente.

    Desvantagens:

    1. Apesar de ser possível ajustar parâmetros de um layout pré-definido, a construção de um novo layout é difícil e toma muito tempo.
    2. É muito difícil escrever documentos sem estrutura e desorganizados.

    O que é UTeX ?

    {Ultima versão lançada em : 09.11.06 11:59 AM }

    O UTeX é um projeto realizado para a Universidade Tecnológica Federal do Paraná (UTFPR) unidade de Cornélio Procópio, e tem como objetivo padronizar as normas técnicas para elaboração de propostas e monografias de conclusão de curso.

    O projeto foi iniciado pelos alunos: {Diogo Cezar Teixeira Batista, Rafael Augusto Ambrosio Rosa} do 7º perído correspondente ao segundo semestre de 2006.

    O UTeX tem como base a utilização da classe ABNTeX.

    Com intúito de se adequar as normas da instituição e em específico as normas do curso de Tecnologia em Informática, deu-se início a construção de um arquivos principal [utfPR.sty] que redefine as principais configurações propostas pelo pacote ABNTeX, ainda nesse arquivo uma série de comandos com prefixo UTFPR foram implementados.

    Além do arquivo de configuração UTeX existe um modelo de trabalho de diplomação mostrando como se utiliza cada um dos comandos redefinidos.

    Fica claro que esse pacote não é destinado a leigos na linguagem LaTeX pois, apesar da intuitivadade dos comandos, não se descarta conhecimentos avançados para elaboração de um bom trabalho.

    ***PEDIMOS ENCARECIDAMENTE***

    Que esse projeto não termine por aqui, que os calouros percebam que LaTeX é indenpendente, livre, organizado, e uma execelente ferramenta para construção de textos tipográficos.

    Por favor calouros, dêm continuidade a esse trabalho, não fizemos isso para nós e sim para vocês.

    Esse pacote também serve para construção de trabalhos acadêmicos.

    Para dúvidas relacionadas ao UTeX (não dúvidas relacionadas ao LaTeX) por favor queiram entrar em contato.
