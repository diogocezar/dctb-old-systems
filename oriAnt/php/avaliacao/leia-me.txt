=> Crie o banco de dados conforme o arquivo bd.sql

=> Mude os dados da conexão (host, user e senha) nos arquivos inserir_avaliacao.php e computar_avaliacao.php

=> Para inserir uma avaliacao em uma pagina apenas acrescente:

    <div id=avaliacao>
        <?php include "computar_avaliacao.php";?>

    </div>

    <div id=comunicacao></div>

    *Colocando esse trecho de código em páginas diferentes o código reconhecerá como avaliações diferentes.
    Se mais de uma avaliação será inserida em uma mesma página, deve-se mudar o parâmetro passado na funcao InserirAvaliacao(), trocando em todos os arquivos a variavel $pg pela chave desejada, que identificará a avaliação como única.

=> Insira o arquivo Javascript do Ajax na página:

     <script type="text/javascript" src="script/ajax.js"></script>
