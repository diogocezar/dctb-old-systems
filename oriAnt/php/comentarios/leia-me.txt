=> Para inserir um sistema de comentários em uma página apenas:
	
	* Insira o arquivo Javascript do Ajax na página:

     <script type="text/javascript" src="script/ajax.js"></script>

	* Acrescente 

    <?php include "../comentario.php"; ?>

    <div id=comunicacao></div>


OBS.:
    *Colocando esse trecho de código em páginas diferentes o código reconhecerá como sistemas de comentários diferentes.
    Se mais de um sistema de comentário será inserido em uma mesma página, deve-se mudar o parâmetro passado na funcao InserirComentario(), trocando em todos os arquivos a variavel $pg pela chave desejada, que identificará o sistema de comentário como único.
